🚀 Get Started in 5 Minutes

Three simple steps to your personal RAG system

1️⃣ Install Ollama

Download from ollama.ai, then run:

ollama pull llama2

2️⃣ Index Your Files

Open Settings → Check: Downloads, Documents, Apple Notes → Click Scan

3️⃣ Start Asking Questions

Go to Chat and try: "What files did I download yesterday?"
